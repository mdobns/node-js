# Next.js Learning Notes

This repository contains my notes and code snippets as I learn Next.js.

## Getting Started

- Install dependencies:  
    ```bash
    npm install
    ```

- Run development server:  
    ```bash
    npm run dev
    ```

## Topics Covered

- Pages and Routing
- Static and Server-side Rendering
- API Routes
- Styling (CSS, Sass, Tailwind)
- Data Fetching
